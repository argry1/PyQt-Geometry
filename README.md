# Геометрические фигуры - Обучающее приложение

Интерактивное приложение на PyQt5, предназначенное для изучения геометрических фигур. Приложение предоставляет информацию о различных фигурах и включает систему тестирования для проверки знаний.

## Возможности

- **Интерактивная информация о фигурах**: Просмотр подробной информации о геометрических фигурах:
  - Треугольник
  - Ромб
  - Трапеция
  - Параллелограмм

- **Система тестирования**: Проверка знаний с помощью вопросов с множественным выбором
- **Профили пользователей**: Введите имя для отслеживания прогресса
- **Подсчет баллов**: Автоматическое начисление баллов за правильные ответы
- **Таблица рекордов**: Просмотр лучших результатов, сохраненных в базе данных SQLite

## Требования

- Python 3.x
- PyQt5
- SQLite3
